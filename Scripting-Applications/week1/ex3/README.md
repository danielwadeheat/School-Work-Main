# Week 1 – Exercise 3: Mini Prompt App

This assignment demonstrates input → clean → output in JavaScript.

- Ask for a name using `prompt()`.
- Clean the input with `trim()`.
- Print a greeting using a template string.

Open `index.html` with Live Server and view the Console output.

## Console Output (Current)

### Example 1
- Input: `"Daniel"`
- Output: `Hello, Daniel!`

### Example 2
- Input: `"  Ava  "`
- Output: `Hello, Ava!`

### Example 3
- Input: `""` (empty) or Cancel
- Output: `Hello, Student!`
