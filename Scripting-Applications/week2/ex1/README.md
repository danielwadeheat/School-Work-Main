# Week 2 – Exercise 1: To-Do List

This assignment demonstrates DOM selection, events, `classList`, and event delegation.

- Add a task from input + Add button.
- Press Enter in the input to add a task.
- Click a task text to toggle `.done`.
- Click Delete to remove a task.
- Event delegation is used on the `<ul>` list.

Open `index.html` with Live Server and view behavior in the browser.

## Current Features

### Add Task
- Input text + Add button (or Enter key)
- Appends a new `<li>` item

### Mark Done
- Click task text to toggle done style

### Delete Task
- Click Delete button to remove item
