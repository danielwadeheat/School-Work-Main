# Week 2 – Exercise 2: Theme Switcher

This assignment demonstrates DOM events, `classList`, and `localStorage`.

- Click button to toggle Light/Dark mode.
- App adds/removes `dark` class on `<body>`.
- Theme choice is saved in `localStorage`.
- Saved theme restores on page load.

Open `index.html` with Live Server and refresh after changing theme.

## Current Behavior

### Toggle
- Light → Dark
- Dark → Light

### Storage
- Key: `preferredTheme`
- Value: `light` or `dark`
