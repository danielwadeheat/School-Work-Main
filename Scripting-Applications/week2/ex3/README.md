# Week 2 – Exercise 3: Live Filter

This assignment demonstrates DOM selection and live filtering.

- Type in the input to filter list items.
- Filtering is case-insensitive.
- Matching text is highlighted using `<mark>`.
- Non-matching items are hidden.

Open `index.html` with Live Server and type in the filter box.

## Current Behavior

### Filter

- Real-time filtering as user types
- Case-insensitive matching

### Highlight

- Matching part of each visible item is wrapped in `<mark>`
