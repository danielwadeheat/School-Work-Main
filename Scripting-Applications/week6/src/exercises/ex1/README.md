# Week 6 – Exercise 1: Catalog Browser

List → Detail app with data, using `HashRouter`, `useEffect`, and loading/empty/error states.

## Notes

- Data source: `public/items.json`
- Search text persists in the URL query string

## Run

- `npm install`
- `npm run dev`

Vite URL:

- List: `http://localhost:5173/#/ex1`
- Detail (example): `http://localhost:5173/#/ex1/item/1`
