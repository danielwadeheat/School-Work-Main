# Week 8 — Exercise 1 (React + Vite)

This exercise uses React + Vite. Run the dev server, then open the exercise page:

- Dev server: `npm run dev`
- URL: `http://localhost:5173/src/ex1/index.html`

## Assignment Requirements
- Add/Edit form with controlled inputs
- Browser validation (`required`, `minLength`, `type`)
- Accessible errors (`aria-invalid`, message with `role="alert"`)
- Disable Save while “saving” (simulated delay)
- On success, return to list view and show a status message

## What This Does
- Displays a list view and a form view
- Uses built‑in HTML validation first
- Shows accessible error messages
- Simulates saving with a timeout and disables the Save button
- Shows “Saved successfully” and returns to the list
