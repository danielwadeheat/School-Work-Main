# Week 8 — Exercise 1 (Standalone)

Open with Live Server (Go Live), or open the `index.html` file in a browser (no Node/Vite needed) or use File → Open. This is a minimal non‑React implementation of the Exercise 1 requirements: add/edit form, built‑in HTML validation (required, minLength, type), accessible error messages, and a simulated save with a disabled button.

This version includes a simple list view, an add/edit form, and a “Saved successfully” status message after the simulated save.
