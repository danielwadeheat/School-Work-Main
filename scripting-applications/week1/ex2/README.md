# Week 1 – Exercise 2: Simple Scoreboard

This assignment demonstrates arrays, objects, loops, and basic logic in JavaScript.

- Players are stored as objects in an array (`name`, `points`).
- `addPoints(playerName, amount)` updates a player's score.
- `getTopPlayer()` finds the player with the highest points.
- A report is printed to the Console.

Open `index.html` with Live Server and view the Console output.
