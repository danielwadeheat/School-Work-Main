# Week 3 – Exercise 1: Users List (GET)

This assignment demonstrates `fetch`, JSON, and `async/await`.

- Fetches users from a public API.
- Shows **Loading...** while waiting.
- Shows a list of names on success.
- Shows **No results** if list is empty.
- Shows a friendly error message on failure.

Open `index.html` with Live Server and check the page + Console.
