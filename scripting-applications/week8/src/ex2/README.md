# Week 8 — Exercise 2 (React + Vite)

This exercise uses React + Vite. Run the dev server, then open the exercise page:

- Dev server: `npm run dev`
- URL: `http://localhost:5173/src/ex2/index.html`

## Assignment Requirements
- Use `loading="lazy"` on images
- Provide width/height for images
- Code-split a route or big component with `React.lazy` + `<Suspense>`

## What This Does
- Renders an image grid with native lazy loading and fixed dimensions
- Demonstrates a small performance‑minded layout
